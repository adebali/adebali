Stable Services

CDvist - Comprehensive Domain Visualation Tool

Protein domain identification is a highly important step in understanding the molecular function of proteins. Various domain search tools have strengths and weakness based on their properties. Some of them are specific and others are sensitive. CDvist combines these tools in a logical pipeline to address the sensitivity problem with a minimum cost of specificity. CDvist is the only web server that provides batch requests (up to 500) for computationally intensive HHM-HMM comparison.

Aquerium - Architecture Querying Podium

One computationl method to reveal or support novel protein-protein interactions is phylogenetic profiling. Cooccurences of proteins are indicators of functional interaction. Aquerium is a web-server to visualize co-presence/absence of any biological marker in genomes. It also provides a database of pre-computed and genome-linked domains to query domain architectures of interest.

SAVER (NPC1) - Single Amino Acid Variant Evaluator

Orthologs and paralogs have different roles in disease. However, current tools do not take this fact into account and do not distinguish them when evaluating the effect of a substitution on health. We think that only orthologs should be considered to have identical function. SAVER is an algorithm that uses several sets (such as orthologs and orthologs after most recent duplication) to evaluate the effects of single amino acid substitutions. This web server shows the results of a test case: effects of missense NPC1 mutations on Niemann-Pick disease type-C. The source code of SAVER is also available.

Examples from my workbench

The tools presented here are just to give idea of what we can do about visualizing the biological data on web. They are neither stable nor guaranteed for a scientific analysis. However, I am open to develop them if I see enough interest. Please contact with me if you feel like they can be useful to you.

BlastCover

One important missing ring in BLAST visualization chain is the coverage display. Knowing how much and what portion of your query actually aligns with the subject (hit) is important. This is an attempt of visualizing coverage of the query and subject. It takes CSV (BLAST results) as an input by default.

OrthologyCrossfilter

In genomics, instant filtering the data may play a crucial role in making sense out of the patterns. This is a javascript crossfilter example on Human-Chimpanzee one-to-one orthologs and some markers of my interests. It is extremely simple to use: it takes CSV as an input. However, it hasn't been generalized to address any CSV yet. Let me know if you are interested in a general version of this tool.

CSVtoHTMLtable

Web-based visualization is becoming more and more glaring. Therefore, although Excel or R are good for tables, they lack in creating custom graphics. HTML SVG helps here. This tool converts your table to HTML by adding another column for custom visualization. In principle any custom graph built based on the column values can be created.

TreeArch

Phylogenetic trees can be enriched in various information. One of the additional information that I wanted to retrieve was domain architectures. We only need to have a json formatted sequence file along with the newick. Domain architectures are retrieved from the SeqDepot database. One deficiency of this approach is the fact that the protein sequences that are not present in SeqDepot cannot be represented with a domain architecture figure.

Treelign

Displaying the alignment and phylogenetic tree together is undoubtedly very handy. All we need is newick and JSON formatted sequence file. Fasta2Json script can be found in my scripts collection in BitBucket.

PhyloBlast

Order of the BLAST results and their phyletic distribution can be useful. This tool enables one by one visualising blast results on a phylogenetic tree and displays scores, e-values and coverage information as well as the domain architecture. Please pay attention where a big gap in the e-value visual is introduced. Can that cutoff be used to determine the homolog sequences?

BlastSnap

This tool shows the blast hits aligned with a phyletic tree. X-axis is the hit number starts with the first hit and ends up with the last hit. I believe, this tool may potentially show horizontal gene transfers, contamination as well as a line of evidence for vertical evolution.

TaxaSunburst

This tool is the basic building block of the Aquerium visualizaiton. Sunburst is a great way to show the taxnomic distriubtion of genomic entitites. Zoomable sunburst was customized based on a d3 example and enabled outer non-hierarchihcal layers.

TaxaCollapsable

Taxonomic hierarchy is coupled by additional information on the domain architectures. This can be useful to identify taxomic lineage specific domain architecture.
